# @openfn/language-pdf

## 1.0.0

### Added

- Initial release of PDF adaptor
- `generatePdf()` function for creating PDF documents from HTML templates and JSON data
- Support for Handlebars templating
- Comprehensive PDF formatting options via Puppeteer
- Support for both inline templates and template files
- Binary buffer and base64 output formats
- Full HTML/CSS support for complex layouts
- Configurable page formats, margins, headers, and footers
- Comprehensive error handling and validation
